# 🚀 OnBoard Framework

Welcome to the **OnBoard** documentation!

OnBoard is a lightweight, modern onboarding and tutorial framework designed for Roblox.

---

## ✨ Features
* 🗺️ **3D World Beams**: Points players toward workspace parts, models, or attachments.
* 👆 **2D UI Pointer**: Animated hand pointer that targets UI elements dynamically.
* 🎛️ **Fully Configurable**: Easily tweak colors, fonts, and textures.
